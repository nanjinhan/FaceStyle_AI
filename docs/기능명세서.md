# 단체 보정 앱 기능명세서 (페이지별)

> v1.0 — 2026-07-18 기준. 이 문서가 최신 명세이며, `backend/README.md`가 언급하는 "공동 보정 앱 기능 명세서 v0.1"을 대체한다.

컬럼 구성: 기능그룹 | 세부 그룹 | 기능명 | 기능 설명 | 입력 | 출력 | 문구 | 정책

## 1. 회원관리

| 기능그룹 | 세부 그룹 | 기능명 | 기능 설명 | 입력 | 출력 | 문구 | 정책 |
|---|---|---|---|---|---|---|---|
| 회원관리 | 회원가입 | 계정 가입 | 이메일/소셜 계정 기반 회원가입 | 가입 버튼 클릭, 계정 정보 입력 | 가입 완료 후 닉네임 설정 화면으로 이동 | "환영해요! 사용할 닉네임을 정해주세요" | 앱 설치 필수, 웹 미지원 |
| 회원관리 | 회원가입 | 닉네임 설정 | 닉네임 입력 (중복 허용) | 닉네임 텍스트 입력 | 닉네임 저장 후 메인으로 이동 | "닉네임은 언제든 바꿀 수 있어요" | 닉네임 중복 허용 |
| 회원관리 | 회원가입 | 고유 색상 배정 | 가입 시 사용자별 고유 컬러 자동 배정 → 커서·프로필·상태 아이콘에 사용 | 가입 완료 | 프로필에 색상 적용 표시 | - | 동일 닉네임 구분자로 사용, 색상은 시스템 자동 배정 |
| 회원관리 | 로그인 | 로그인 | 계정 기반 로그인 | 로그인 버튼 클릭 | 메인페이지로 이동 | - | 로그인 세션 유지 |
| 회원관리 | 로그인 | 게스트 참여 | 초대 링크 진입 시 로그인 없이 닉네임만 입력하고 앨범/방 참여 | 초대 링크 클릭 → 닉네임 입력 | 해당 방/앨범으로 입장 | "닉네임만 정하면 바로 들어갈 수 있어요" | 게스트 데이터는 기기 기준, 앱 삭제 시 복구 불가 안내 |
| 회원관리 | 계정 | 로그아웃 / 탈퇴 | 로그아웃 및 회원 탈퇴 | 마이페이지 내 버튼 클릭 | 로그인 화면으로 이동 | 탈퇴 시 "보정 기록과 앨범 접근 권한이 사라져요" | 탈퇴 시 본인이 방장인 앨범은 위임 또는 삭제 선택 필수 |

## 2. 메인페이지 (홈)

| 기능그룹 | 세부 그룹 | 기능명 | 기능 설명 | 입력 | 출력 | 문구 | 정책 |
|---|---|---|---|---|---|---|---|
| 메인페이지 | 전체 화면 | 홈 화면 | 로그인 후 노출되는 메인 화면. 내 앨범 목록 + 최근 작업 사진 표시, 무한 스크롤 | 로그인 완료 | 앨범 카드 리스트 + 최근 보정 사진 | - | - |
| 메인페이지 | 메인 화면 | 할 일 배너 | 내 얼굴이 있는데 아직 보정 안 한 사진 개수 표시 | 홈 진입 | "보정 대기 N장" 배너 노출 | "친구들이 기다리고 있어요! 보정할 사진 N장" | 대기 사진 0장이면 배너 미노출 |
| 메인페이지 | 메인 화면 | 앨범 바로가기 | 앨범 카드 클릭 시 해당 앨범으로 이동 | 앨범 카드 클릭 | 앨범 상세 페이지 이동 | - | - |
| 메인페이지 | 방 생성 | 실시간 방 만들기 | 홈에서 실시간 협업 방 즉시 생성 | "같이 보정하기" 버튼 클릭 | 방 생성 + 초대 링크 화면 이동 | "링크를 보내서 친구를 초대하세요" | - |
| 메인페이지 | 촬영 | 카메라 진입 | 홈에서 인앱 카메라 바로 진입 | 카메라 버튼 클릭 | 촬영 화면 이동 | - | - |
| 메인페이지 | 알림 | 알림 목록 | 업로드/완료/초대 등 알림 모아보기 | 알림 아이콘 클릭 | 알림 리스트 표시 | - | 알림 클릭 시 해당 사진/앨범으로 이동 |

## 3. 실시간 방

| 기능그룹 | 세부 그룹 | 기능명 | 기능 설명 | 입력 | 출력 | 문구 | 정책 |
|---|---|---|---|---|---|---|---|
| 실시간 방 | 초대 | 링크 초대 | 초대 링크 생성 및 공유. 링크 클릭 시 웹이 아닌 앱으로 딥링크 연결 | 초대 버튼 클릭 → 링크 공유 | 상대방 앱 실행 후 방 자동 입장 | "OO님이 같이 보정하자고 초대했어요" | 미설치 시 스토어 이동 → 설치 후 방 자동 입장 |
| 실시간 방 | 초대 | 입장 인원 표시 | 방장·친구1·친구2 등 참여자를 닉네임 + 고유 색상 동그라미로 표시 | 방 입장 | 상단에 참여자 아바타 노출 | - | 방 인원 제한 6명 (렉 방지) |
| 실시간 방 | 사진 선택 | 컷 선택 | 여러 장 업로드 후 보정할 컷 함께 선택 | 사진 다중 업로드 → 컷 투표/선택 | 선택된 컷으로 에디터 진입 | "어떤 컷으로 보정할까요?" | - |
| 실시간 방 | 클레임 | 내 얼굴 지정 | 사진 속 인물 자동 인식 → 각자 "내 얼굴" 선택해 본인 영역 지정 | 얼굴 영역 탭 → "이게 나예요" 클릭 | 해당 얼굴에 본인 색상 테두리 표시 | "내 얼굴을 선택해주세요" | 얼굴 1개당 1명만 클레임 가능, 클레임 해제는 본인만 |
| 실시간 방 | 권한 | 편집 권한 분리 | 클레임한 얼굴/몸 영역은 본인만 편집 가능, 타인 영역은 잠금 | 타인 영역 터치 시 | 잠금 아이콘 + 편집 불가 표시 | "OO님의 영역이에요" | 핵심 원칙: 내 것만 편집 |
| 실시간 방 | 권한 | 공용 영역 권한 | 배경·전체 톤 등 공용 영역 편집 권한 설정 | 방장 설정 메뉴에서 선택 | 방장 전용 / 전체 허용 적용 | - | 기본값: 방장 전용 |
| 실시간 방 | 동시 편집 | 실시간 커서 | 각 참여자가 편집 중인 위치를 색상 커서로 실시간 표시 | 참여자의 화면 조작 | 색상 동그라미 커서 이동 표시 | - | 렉·끊김 없이 동기화, 동시 편집 시 결과물 깨짐 방지 |
| 실시간 방 | 동시 편집 | 작업 상태 표시 | 누가 어떤 기능을 쓰는 중인지 표시 | 참여자의 기능 선택 | "민지 — 잡티 제거 중" 라벨 표시 | - | - |
| 실시간 방 | 되돌리기 | 개인별 실행취소 | 실행취소/다시하기는 본인 편집 내역 기준으로만 작동 | 실행취소 버튼 클릭 | 본인 마지막 작업만 되돌림 | - | 타인 작업에 영향 없음 |
| 실시간 방 | 되돌리기 | 기능별 리셋 | 전체 초기화가 아닌 항목별 리셋 (예: 내 주름 보정만 초기화) | 항목 선택 → 리셋 클릭 | 해당 항목만 원본으로 복원 | "이 항목의 보정만 초기화할까요?" | 본인 영역 항목만 리셋 가능 |
| 실시간 방 | 저장 | 개인 저장 | 각자 원할 때 현재 공유 상태를 개인 갤러리에 저장 | 저장 버튼 클릭 | 기기 갤러리에 저장 완료 토스트 | "내 갤러리에 저장했어요" | 타인 작업/방 상태에 영향 없음 |
| 실시간 방 | 저장 | 완료 확정 | 전원이 확정 체크 시 최종본 확정 + 전원 갤러리 자동 저장 | 각자 "완료" 체크 | 전원 완료 시 최종본 확정 알림 | "모두 완료! 최종본이 저장됐어요" | 확정 후 편집 잠금, 결과물은 앨범에도 자동 보관 |
| 실시간 방 | 종료 | 방 자동 만료 | 실시간 방은 일정 시간 후 자동 종료 | 24시간 경과 | 방 종료, 작업본은 앨범으로 이관 | "방이 종료되어 앨범에 저장했어요" | 서버 부담·프라이버시 관리 목적 |

## 4. 앨범 (비동기 협업)

| 기능그룹 | 세부 그룹 | 기능명 | 기능 설명 | 입력 | 출력 | 문구 | 정책 |
|---|---|---|---|---|---|---|---|
| 앨범 | 생성 | 그룹 앨범 생성 | 친구 그룹 단위 앨범 생성, 모임별 다수 운영 가능 | 앨범 만들기 → 이름 입력 | 앨범 생성 완료 | "앨범 이름을 정해주세요 (예: 제주 여행)" | 생성자가 방장 권한 자동 획득 |
| 앨범 | 멤버 | 멤버 초대 | 초대 링크로 멤버 추가 | 초대 링크 공유 | 상대방 앱으로 입장 | - | 초대 권한은 방장만 보유 |
| 앨범 | 멤버 | 멤버 나가기 | 멤버 스스로 앨범 탈퇴 | 나가기 버튼 클릭 | 앨범 목록에서 제거 | "앨범에서 나가면 다시 초대받아야 해요" | 본인 의사로 언제든 가능 |
| 앨범 | 멤버 | 멤버 내보내기 | 방장이 특정 멤버 강제 퇴장 | 방장이 멤버 선택 → 내보내기 | 해당 멤버 접근 차단 | - | 방장 전용 권한 |
| 앨범 | 멤버 | 방장 위임 | 방장 권한을 다른 멤버에게 넘김 | 방장이 멤버 선택 → 위임 | 방장 권한 이전 | "방장 권한을 넘길까요?" | 앨범 삭제 전 위임 가능 |
| 앨범 | 관리 | 앨범 삭제 | 그룹 앨범 자체 삭제 | 방장이 삭제 클릭 → 확인 | 앨범 및 사진 삭제 | "삭제하면 되돌릴 수 없어요" | 방장 전용 권한 |
| 앨범 | 업로드 | 사진 업로드 | 멤버 누구나 앨범에 사진 업로드 | 업로드 버튼 → 사진 선택 | 앨범에 사진 추가 | - | - |
| 앨범 | 업로드 | 업로드 알림 | 새 사진 업로드 시 앨범 멤버 전체 푸시 알림 | 사진 업로드 완료 | 멤버 전원에게 푸시 발송 | "수진님이 사진 3장을 올렸어요" | 앨범별 알림 on/off 설정 가능 |
| 앨범 | 조회 | 진행 상태 아이콘 | 사진별 대기중/보정중/완료 상태 아이콘 표시 | 앨범 진입 | 사진 썸네일에 상태 뱃지 (예: 2/3 완료) | - | 사진 속 클레임 인원 기준으로 계산 |
| 앨범 | 조회 | 할 일 정렬 | 내 얼굴이 있는데 내가 아직 안 한 사진만 모아보기 | 필터 선택 | 해당 사진만 리스트 표시 | "내가 보정할 사진" | 정렬: 최신순/할일순 |
| 앨범 | 보정 | 비동기 보정 | 각자 편한 시간에 들어가 본인 영역만 보정 (클레임 방식 동일) | 사진 선택 → 에디터 진입 | 본인 영역 편집 화면 | - | 실시간 방과 동일한 권한 정책 적용 |
| 앨범 | 확정 | 완료 자동 확정 | 사진 속 전원 완료 시 자동으로 최종본 확정 + 알림 | 마지막 멤버 완료 체크 | 최종본 확정 + 전원 알림 | "모두 완료! 최종본이 확정됐어요" | 확정 후 편집 잠금 |
| 앨범 | 확정 | 방장 마감 | 방장이 미완료 사진을 강제 확정 | 방장이 마감 클릭 → 확인 | 현재 상태로 최종본 확정 | "아직 안 한 친구가 있어요. 지금 마감할까요?" | 한 명이 장기간 미완료 시 대비 |
| 앨범 | 삭제 | 사진 삭제 | 앨범 내 사진 삭제 | 사진 선택 → 삭제 클릭 | 앨범에서 사진 제거 | "삭제하면 복구할 수 없어요" | 업로더 본인 + 방장만 가능 |
